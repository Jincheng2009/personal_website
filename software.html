SQLEngine: Realization of SQL function in Java environment (Java, JavaCC and SQL)


Programming languages involved: Java, JavaCC, TPC-H, makefile, SQL

Phase I: JavaCC to parse SQL statement and Java program to execute SQL statement.
Phase II: Build ISAM and Hash index under memory restriction (BufferManager API was provided by the instructor limiting the memory size).
Phase III: Generate large-scale database by TPC-H and optimize corresponding SQL trees to accelerate SQL execution: (1) Pushdown selection predicate and Projection operator; (2) Realize merge-sort join instead of nested loop join and (3) Utilize index built on phase II to accelerate scan operation.
